# FreeClaw

Minimal, unrestricted agent runtime.

## Scope
- Providers: OpenAI, Anthropic
- Channel: Telegram
- Tools: exec, read, write, fetch
- Execution: root-first (`requireRoot: true`)

## Quick start
```bash
cp freeclaw.config.example.json freeclaw.config.json
# set keys/tokens
npm run start
```
