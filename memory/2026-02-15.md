## OAuth 2.0 Token Refresh (Evening Discovery)

### The Problem
OAuth 2.0 tokens expire after 2 hours (7200 seconds). Token created at 13:28 UTC expired around 15:28 UTC. Couldn't read mentions/search results after expiry.

### The Solution - Manual Refresh
```python
import requests
import json
from datetime import datetime

CLIENT_ID = "NV9Zc3B3bTc0T1l2TEpFb0RyTjQ6MTpjaQ"
REFRESH_TOKEN = "[REDACTED_REFRESH_TOKEN]"

response = requests.post(
    "https://api.twitter.com/2/oauth2/token",
    auth=(CLIENT_ID, ""),
    data={
        "grant_type": "refresh_token",
        "refresh_token": REFRESH_TOKEN,
        "client_id": CLIENT_ID
    }
)

if response.status_code == 200:
    token_data = response.json()
    token_data['created_at'] = datetime.now(datetime.UTC).isoformat()
    
    with open('/root/.config/corvus/credentials/x_oauth2.json', 'w') as f:
        json.dump(token_data, f, indent=2)
    
    print("✅ Token refreshed!")
```
